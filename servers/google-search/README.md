## Install

### Quick Start (recommended)

```bash
BEARER_TOKEN=YOUR_BEARER_TOKEN \
uvx mcparmory-google-search
```

### With pip

```bash
pip install mcparmory-google-search
BEARER_TOKEN=YOUR_BEARER_TOKEN \
mcparmory-google-search
```

### MCP Client Configuration

Add to your MCP client config (e.g. Claude Desktop, Cursor, Codex):

```json
{
  "mcpServers": {
    "google-search": {
      "command": "uvx",
      "args": ["mcparmory-google-search"],
      "env": {
        "BEARER_TOKEN": "YOUR_BEARER_TOKEN"
      }
    }
  }
}
```

## Credentials

Set the following environment variables (via MCP client `env` config, shell export, or `.env` file):

- `BEARER_TOKEN` — Bearer token

## Run Locally

**First**, configure your credentials in `.env` (see [Credentials](#credentials) above).

```bash
pip install -r requirements.txt
python server.py
```

## Connect MCP Client

Edit `.mcp.json` and replace `<SERVER_DIR>` with the absolute path to this directory, then add to your MCP client configuration.

Example (if server is at `/home/user/mcp-servers/google-search`):
```json
{
  "mcpServers": {
    "google-search": {
      "command": "python",
      "args": ["/home/user/mcp-servers/google-search/server.py"]
    }
  }
}
```

## Docker

### Pre-built image (recommended)

```bash
docker run -p 8000:8000 \
  -e BEARER_TOKEN=YOUR_BEARER_TOKEN \
  ghcr.io/mcparmory/google-search:latest
```

### Build from source

**First**, configure your credentials in `.env` (see [Credentials](#credentials) above).

```bash
docker build -t google-search .
docker run -p 8000:8000 --env-file .env google-search
```

**Before running**, make sure ports 8000 are free.### MCP client config (Docker)

For Docker, use SSE transport in your MCP client config:
```json
{
  "mcpServers": {
    "google-search": {
      "type": "sse",
      "url": "http://localhost:8000/sse"
    }
  }
}
```
